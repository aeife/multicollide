\section{Verwendete Software}
\label{sec:Verwendete Software}
Im Folgenden werden die Programme vorgestellt, welche zur Erstellung dieser Arbeit und Entwicklung des Produktkonfiguratorplugins benutzt wurden.

\begin{itemize}
	\itemd{Dia}{Sämtliche Diagramme wurden mit dem Zeichenprogram Dia 0.97.1 erstellt.}
	\itemd{PHPEdit}{Der gesamte Quellcode des Produktkonfiguratorplugins wurde mit dem PHP Editor PHPEdit in der Version 3.2.0.8595 geschrieben.}
	\itemd{XAMPP}{Zum Testen des Plugins auf einem lokalen Computer wurde \gls{XAMPP} verwendet. Dabei musste auf die Version 1.6.7. zurückgegriffen werden um Kompatibilitätsprobleme mit xt:Commerce Veyton und dem benötigten IonCube Loader zu vermeiden.}
	\itemd{FileZilla}{Für den Transfer der Daten auf den Webserver des Onlineshops wurde die Version 3.2.7.1 des FTP-Client FileZilla benutzt.}
	\itemd{Firefox}{Die Verwaltung des Shopsystems sowohl im \gls{glos:Frontend} als auch im \gls{glos:Backend} wurde in dem Browser Firefox erledigt. Das Backend von xt:Commerce Veyton ist darüber hinaus auch lediglich in Firefox fehlerfrei darstellbar.}
	\itemd{\LaTeX}{Zur Erstellung dieser Arbeit wurde {\LaTeX} verwendet. Als Distribution wurde MiKTeX 2.9 verwendet und als Editor die Software TeXnicCenter sowie TextMate.}
	\itemd{JabRef}{Zur Verwaltung des Literaturverzeichnisses dieser Arbeit wurde die Software JabRef 2.6 verwendet.}
\end{itemize}
