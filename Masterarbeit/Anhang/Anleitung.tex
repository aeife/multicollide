\section{Anleitung}
\label{sec:Anleitung}

Anleitung zur Installation und Benutzung des Plugins im Shopsystem xt:Commerce Veyton.

\subsection{Installation}
Für die Installation des Produktkonfigurators kopieren Sie den mitgelieferten Ordner \code{pc\_product\_configurator} in den Ordner \code{plugins} im Hauptverzeichnis Ihres Onlineshops befindet. In der Datei \code{product.html}, welche sich im Pfad \code{templates/xt\_default/xtCore/pages/product/} Ihres Onlineshops befindet, muss zudem noch der Code \code{\{hook key=pc\_product\_configurator\}} vor der Zeile \code{<div class="'addtobasket"'>} eingefügt werden.\\
Anschließend melden Sie sich im Backend Ihres Shops an und gehen zu dem Menüpunkt Inhalte / Plugins / deinstallierte Plugins. Dort sehen Sie unter der Modulklasse "`products"' den Produktkonfigurator. Klicken Sie auf das grüne Symbol rechts und bestätigen Sie die Installation. Bei Erfolg der Installation erhalten Sie eine entsprechende Meldung, welche wiederum bestätigt werden muss.\\
Das Plugin ist nach der Installation zunächst noch inaktiv. Um es zu aktivieren gehen Sie unter Inhalte / Plugins / installierte Plugins. Dort sollten Sie den Produktkonfigurator in der Liste sehen. Ist dies nicht der Fall klicken sie auf "`neu laden"'. Bearbeiten Sie nun den Produktkonfigurator indem Sie zum Beispiel doppelt auf die entsprechende Zeile klicken. Im neu erscheinenden Fenster setzen Sie einen Haken bei "`Status"' und speichern die Bearbeitung. Der Produktkonfigurator ist nun aktiv und kann von Ihnen genutzt werden.

\subsection{Generelle Funktionsweise}
Um ein grobes Verständnis im Umgang mit dem Produktkonfigurator zu erlangen soll an dieser Stelle ein grober Überblick über die Funktions- und Ablaufweise gegeben werden.\\
Der Produktkonfigurator ermöglicht es Produkten Gruppen von Komponenten zuzuweisen. Jede zugewiesen Gruppe entspricht einem Auswahlfeld auf der jeweiligen Produktseite im Shop. Diese Auswahlfelder beinhalteten zunächst die Standardkomponenten, sofern diese Gruppen eine definiert haben. Ist keine Standardkomponente für eine Gruppe definiert bedeutet das, dass die Gruppe optional ist. Zur Auswahl stehen jeweils alle Komponenten, welche die Gruppe beinhaltet. Um die Gruppen besser gliedern zu können werden sie in übergeordnete Gruppen eingeteilt.\\ 
Der Ablauf besteht also darin zunächst Hauptgruppen- bzw. Kategorien anzulegen. In diesen Hauptgruppen können Untergruppen angelegt werden. Dies entspricht den Gruppen, welche später Produkten direkt zugewiesen werden können. Untergruppen beinhalten dann die einzelnen Komponenten, von denen optional eine als Standardkomponente deklariert werden kann.\\
Im Folgenden werden die einzelnen Schritte und deren Durchführung näher erläutert.

\subsection{Komponentengruppen verwalten}
Für die Verwaltung Ihrer Komponentengruppen ist ein gleichnamiger Menüpunkt in der Shopsektion vorhanden. Nach einem Klick auf diesen Menüpunkt befinden Sie sich zunächst in der Ansicht für die Hauptkomponentengruppen. Hier können Sie neue Hauptgruppen anlegen oder vorhandene Hauptgruppen bearbeiten bzw. löschen. Ebenso können sie Hauptgruppen kopieren, wobei sämtliche enthaltene Inhalte wie Untergruppen und Komponente mitkopiert werden.\\
Um zur Ansicht für die Untergruppen einer bestimmten Hauptgruppe zu gelangen klicken Sie entweder in der Bearbeitenansicht unten auf den Button "`Komponentenuntergruppen"' oder in der Übersicht rechts von der entsprechenden Gruppe auf das Ordnersymbol. Sie befinden sich daraufhin in einer Übersicht, welche alle Untergruppen der jeweiligen Hauptgruppe anzeigt. Auch hier können Sie neue Untergruppen anlegen oder vorhandene Untergruppen bearbeiten bzw. löschen.  Eine Besonderheit stellt hierbei das Kopieren dar. Das einfache Kopieren kopiert die Untergruppe innerhalb der jeweiligen Hauptgruppe. Es dupliziert die Gruppe quasi an der aktuellen Position. Das Kopieren von Untergruppen in Hauptgruppen bewirkt hingegen, dass sich ein extra Fenster öffnet. Dieses zeigt Ihnen eine Baumstruktur aller von Ihnen angelegten Haupt- und Untergruppen. Sie können hier die Hauptgruppen auswählen, in welche die Untergruppe kopiert werden soll. Dabei werden wie beim normalen Kopieren sämtliche enthaltene Komponente mitkopiert.

\subsection{Komponenten verwalten}
Um die einzelnen Komponenten einer Untergruppe zu bearbeiten lassen Sie sich zunächst, wie beschrieben, die Untergruppen einer gewünschten Gruppe anzeigen. In dieser Ansicht klicken Sie auf den Button "`Komponenten"'. Sie finden diesen Button erneut entweder unten in der Bearbeitenansicht   oder in der Übersicht rechts von der entsprechenden Gruppe als Bausteinsymbol. Nach einem Klick auf einen der beiden Buttons wird Ihnen die Komponentenübersicht angezeigt. Hier sehen Sie alle Komponenten der jeweiligen Untergruppe. Sie können neue Komponenten anlegen oder die bereits bestehenden verwalten. Neben dem Bearbeiten und Löschen von Komponenten stehen Ihnen wie bei den Untergruppen 2 Arten von Kopieren zur Verfügung. Das einfache Kopieren kopiert die Komponente innerhalb der eigenen Untergruppe, dupliziert diese also. Das Kopieren einer Komponente in Untergruppen bewirkt, dass sich ein Fenster mit einer Baumhierarchie Ihrer angelegten Gruppen und Untergruppen öffnet. Hier können Sie alle Untergruppen anwählen, in welche die Komponente kopiert werden soll.\\
Ein weiterer wichtiger Punkt bei der Verwaltung der Komponenten stellt die Auswahl der Standardkomponenten dar. Sie repräsentiert die Komponente, welche im Shop von Anfang an ausgewählt ist, also zur Standardausstattung des jeweiligen Produktes gehört. Gibt es in einer Gruppe keine Komponente, welche als Standardkomponente definiert ist, gilt die Gruppe als optional. Im Shop ist zunächst bei dieser Gruppe keine Komponente gewählt, aber es kann optional eine der enthaltenen Komponenten ausgewählt werden.

\subsection{Einem Produkt Komponentengruppen zuweisen}
Jedem im Shop verfügbaren Artikel können Komponentengruppen zugewiesen werden. Dabei sind immer konkrete Untergruppen, mit deren enthaltenen Komponenten, gemeint. Um dies zu bewerkstelligen lassen Sie sich im Backend Ihres Shops das jeweilige Produkt entweder in der Bearbeitenansicht oder in der Übersicht anzeigen. In beiden Ansichten befindet sich bei aktivierten Produktkonfigurator ein Button "`Komponentengruppen zuweisen"'. Dies öffnet ein Fenster, in dem alle Komponentengruppen und deren Untergruppen hierarchisch dargestellt sind. Um eine Untergruppe dem Produkt zuzuweisen, setzen Sie an die jeweilige Gruppe einen Haken und bestätigen Sie Ihre Eingabe mit einem Klick auf "`speichern"'. Dem Produkt sind nun die jeweilig ausgewählten Gruppen zugewiesen und stehen dem Nutzer nun in Ihrem Shop zur Verfügung. Sie können jederzeit die Zuweisung der Gruppen durch erneutes Öffnen des Fensters verändern und somit Gruppen hinzufügen oder entfernen.\\
Damit in Ihrem Shop der Produktkonfigurator zu dem jeweiligen Produkt auch angezeigt wird, müssen Sie in den Eigenschaften des Produkts noch einen Haken bei "`Produktkonfigurator"' setzen. Damit ist der Konfigurator für das jeweilige Produkte aktiviert und kann in Ihrem Shop genutzt werden.
